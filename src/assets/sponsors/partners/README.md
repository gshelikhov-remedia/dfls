# Loghi Aziende Sponsor 2026

Drop each company's logo here. The file is picked up automatically by
`src/data/content.ts` (`partnerLogo()`) — no code change needed. Until a file
exists, the card shows the company name as text instead.

Match the base filename exactly (any extension: `.svg`, `.webp`, `.png`, `.jpg`).
Prefer `.svg`, otherwise a transparent-background `.webp`/`.png`.
